1. Внешние API и сервисы

  • Firebase (Google) — аутентификация и хранение конфигурации: firebase.google.com
  • Jolpica / Ergast API — результаты гонок: api.jolpi.ca/ergast/f1

2. Изображения и медиа

  • FlagCDN — изображения флагов: flagcdn.com
