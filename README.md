# 🏎 F1 BINGO — Сезон 2026

Неофициальная интерактивная игра-бинго по Формуле 1 сезона 2026.  
Отмечай события гонок, собирай бинго и следи за всем календарём прямо в браузере.

## Демо

http://f1bingo.ru/

---

## Возможности

### Два режима игры
- **Бинго Гонки (3×3)** — 9 случайных событий одной гонки. Собери линию и получи BINGO!
- **Бинго Сезона (5×10)** — карточка из 50 событий на весь сезон. Закрывай строки и столбцы по ходу чемпионата.

### Аутентификация
- Вход через **Google** (OAuth)
- Вход и регистрация по **email / паролю**
- Авторизация на Firebase Authentication

### Календарь F1 2026
- Все 24 Гран-При сезона с флагами, датами и значками спринт-уикендов
- Тултип с кратким описанием трассы при наведении
- Подробная страница каждого этапа:
  - схема трассы, статистика (число кругов, длина, дистанция, год первого ГП, рекорд круга)
  - интересные факты о трассе
  - результаты гонки / квалификации / спринта в реальном времени через **Jolpica / Ergast API**

### База событий
- 100+ уникальных событий: победы пилотов, аварии, стратегии, штрафы, курьёзы
- События можно настраивать через **панель администратора**: добавлять, удалять, включать для нужного режима, задавать количество чекбоксов (события, требующие подтверждения N раз)

### Статистика
- Количество сыгранных партий и полученных бинго
- Рекорд отмеченных клеток
- Текущая сессия

### Прогресс сохраняется
- Состояние карточки хранится в `localStorage` — всё на месте после перезагрузки страницы

| Слой | Инструмент |
|------|-----------|
| Разметка | HTML5 |
